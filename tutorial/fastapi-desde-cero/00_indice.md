# 00 - Indice del curso

## Objetivo

Aprender FastAPI desde cero hasta construir una API con:

- Endpoints REST
- Validacion de datos
- CRUD
- Base de datos SQLite
- Autenticacion basica con JWT

## Ruta de aprendizaje

1. Preparar entorno
2. Primera app y Swagger
3. Rutas y parametros
4. Modelos con Pydantic
5. CRUD en memoria
6. Persistencia con SQLModel
7. Seguridad basica
8. Siguientes pasos para produccion
