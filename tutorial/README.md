## Requisitos

- Python 3.10 o superior
- Terminal (zsh, bash, PowerShell o similar)

## Forma recomendada de uso

1. Lee cada archivo en orden.
2. Copia y ejecuta los comandos.
3. Prueba los endpoints en `http://127.0.0.1:8000/docs`.
4. Haz los ejercicios al final de cada modulo.
