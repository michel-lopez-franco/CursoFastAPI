# Manual FastAPI: De Cero a Pro (Nivel Principiante)

Este manual está diseñado para aprender FastAPI con calma, entendiendo cada línea y el motivo técnico detrás.

## Enfoque del manual

Cada tema sigue la misma estructura:

1. Introducción y concepto.
2. Escenario de aplicación real.
3. Código por fragmentos pequeños con explicación línea por línea.
4. Código completo final unificado y comentado.
5. Prácticas (2 retos) + archivo de soluciones sugeridas.

## Cómo usar esta carpeta

1. Abre un tema principal (`XX_*.md`).
2. Estudia y ejecuta los bloques pequeños en orden.
3. Resuelve los ejercicios sin mirar la solución.
4. Revisa el archivo `*_soluciones.md` para comparar.
